MiniZinc Resources

The MiniZinc Handbook

The MiniZinc Handbook contains all the documentation for the MiniZinc language and system. It includes:

  • The MiniZinc Tutorial, a step-by-step introduction to modelling with MiniZinc;
  • The MiniZinc User Manual, which covers the command line tools, the MiniZinc IDE and some additional tools; and
  • The MiniZinc Reference Manual, with the official specifications of MiniZinc, FlatZinc, and the MiniZinc library of global constraints and built-in functions.

The MiniZinc Handbook is also available as a printable PDF.
