- Xpress solver migration from BCL to C API with support for Xpress 9.8
  parameters. Add support for indicator constraints, warm start, and
  convex quadratic inequalities.
- The compiler now removes all non-array right-hand sides from variable
  declarations in FlatZinc. This minimizes the effort required when
  parsing `.fzn` or FlatZinc JSON files.
- Add support for SCIP 10, and remove support for SCIP 7 and earlier
  (:bugref:`999`).
- Add versions of `count_*` constraints for booleans, floats and sets of
  integers to allow solvers to support them natively (:bugref:`1005`).
- Add the `--cleanup-time-limit` option for configuring the grace period for
  solver cleanup before forced termination (:bugref:`994`).
- Plus 15 bugfixes!
