MiniZinc 2

Installation Instructions for Linux (binary)

These instructions are for installing MiniZinc 2 from a binary distribution for 64 bit Linux. Download the installer here:

  • Linux: minizinc-{{ mznversion }}-linux64.tar.gz

To install, simply unpack the archive anywhere. After unpacking, add the bin directory inside the unpacked directory to the PATH.

You can now run minizinc, mzn2fzn, flatzinc, and solns2out.

You can also find documentation and examples in the doc directory.

Using MiniZinc 2.0 with the MiniZinc IDE

In order to use MiniZinc 2 with the MiniZinc IDE, you need to change the MiniZinc path in the preferences dialog to point to the bin where the MiniZinc binaries can be found.
