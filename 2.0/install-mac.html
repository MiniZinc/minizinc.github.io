MiniZinc 2

Installation Instructions for Mac OS

These instructions are for installing MiniZinc 2 from a binary distribution for Mac OS.

Note: This package only contains the MiniZinc compiler, you will need to install FlatZinc solvers manually. For most users, downloading the bundled packages is the preferred option!

Download the installer here:

  • Mac OS installer: Install.MiniZinc.{{ mznversion }}.pkg

After downloading, right-click the package and select "Open". This step is necessary because we currently cannot provide a digital signature for the installation package. Follow the instructions in the installer.

Alternatively, if you prefer not to install MiniZinc globally, you can download the followin binary package and simply unpack it anywhere:

  • Mac OS binary archive: minizinc-{{ mznversion }}-Darwin.tar.gz

Running MiniZinc

The installer will place the MiniZinc executables in /usr/local/bin, which is on the standard system path. From a terminal, you can now run minizinc, mzn2fzn, flatzinc, and solns2out.

You can also find documentation and examples in /usr/local/share/doc/minizinc/.

Using MiniZinc 2.0 with the MiniZinc IDE

In order to use MiniZinc 2 with the MiniZinc IDE, you need to change the MiniZinc path in the preferences dialog to point to the /usr/local/bin, where the MiniZinc binaries can be found.
